# Лабораторные работы по программированию
## Описание репозитория
Данный репозиторий содержит лабораторные работы, выполненные в рамках курса по программированию. В ходе выполнения лабораторных работ были изучены и реализованы следующие темы:
1. Ввод/вывод и форматирование.
2. Коллекции и матрицы (list/tuple/set/dict).
3. Тексты и частоты слов (словарь/множество).
4. Работа с файлами CSV и XLSX.
5. Работа с файлами JSON и CSV.
6. CLI‑утилиты с argparse (cat/grep‑lite + конвертеры)
7. Тестирование: pytest + стиль кода (black).
8. ООП в Python: @dataclass Student, методы и сериализация.
9. «База данных» на CSV: класс Group, CRUD-операции и CLI.
10. Структуры данных: Stack, Queue, Linked List и бенчмарки.

## Структура репозитория
Каждая лабораторная работа оформлена в отдельной папке внутри директории `src`. В каждой папке содержатся следующие элементы:
- `README.md`: Отчет по лабораторной работе, содержащий описание задания, требования и примеры запуска.
- Модули с реализацией функций и классов, соответствующих заданию.
- Скриншоты, иллюстрирующие результаты выполнения заданий и примеры работы кода.
